# Weather App

## Overview
The Weather App is a simple Python script that provides current weather information for a given city using the OpenWeatherMap API. It includes features such as temperature (in Celsius and Fahrenheit), feels-like temperature, wind speed, humidity, general weather description, sunrise, and sunset times.

## Requirements
- Python 3.x
- `requests` library
- OpenWeatherMap API key

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/
   cd WeatherApp
   ```

2. Install required Python library:
   ```bash
   pip install requests
   ```

3. Obtain an OpenWeatherMap API key:
   - Sign up for a free account on [OpenWeatherMap](https://openweathermap.org/).
   - Create an API key in your account dashboard.

4. Save your API key in a file named `API.txt` in the project directory.

## Usage
1. Run the script:
   ```bash
   python main.py
   ```

2. Enter the name of the city for which you want to check the weather.

3. To exit the app, type 'exit' when prompted for the city.

4. Optionally, you can check weather data for another city by typing 'yes' when prompted.

## Note
- Ensure a stable internet connection for fetching weather data.
- The OpenWeatherMap API key is sensitive information. Keep it secure and do not share it publicly.

## Files
- `main.py`: The main Python script.
- `API.txt`: File to store your OpenWeatherMap API key.
